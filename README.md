# Welcome To Course Builder

## To set up project
    - clone the project on your local

### For Backend-set-up
    - Execute `cd backend`
    - Execute `npm install`
    - Create a `.env` file in the root directory and add the following environment variable.
        - `PORT = 3000`
        - `MONGO_DB_URL= mongodb://127.0.0.1:27017/<YOUR_DATABASE_NAME>`

## DB Design
    - Course model
    - User model
    - Payment model
    - Statics model
